# Aryan Raj | Portfolio

Modern portfolio website focused on AI, ML, and full-stack development.

Live site: https://wdaryan.github.io/Portfolio

## Highlights

- Responsive UI for desktop, tablet, and mobile
- Admin panel to update portfolio sections quickly
- Dynamic sections powered by browser storage
- Projects, certificates, blog, testimonials, and contact
- Experience section with internship timeline

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub Pages for deployment

## Sections Included

- Home
- About
- Skills
- Projects
- Work Experience and Internships
- Certificates
- Blog
- Reviews
- Contact

## Admin Access

Admin URL: /admin.html

The admin panel allows you to add, edit, and delete:

- Hero content
- About content
- Skills
- Projects
- Experience entries
- Blog posts
- Certificates

## Local Development

1. Clone the repository.
2. Open the folder.
3. Run index.html in your browser.
